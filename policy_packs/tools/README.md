# Offline Assessment Tools

Standalone HTML assessment tools are temporarily removed from this pack while
the main PallasGuard workflow becomes the source of truth.

This directory is kept as a placeholder for future downloadable helpers.

## Customization

To adapt for your organization later:
- Clone this framework and create a new folder under `frameworks/` using `framework-template/` as a scaffold.
- Replace generic option labels (e.g., "Organization Cloud", "Business Unit A") with your organization's actual values.
- Update `manifest.yaml` with your pack `id` and brand settings.
